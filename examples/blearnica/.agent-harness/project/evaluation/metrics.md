# Metrics

## Product Metrics

- Activation: percentage of users who start and complete a diagnostic test.
- Retention: percentage of users who return within 7 days.
- Completion: percentage of users who complete at least one learning path step.
- Quality: percentage of feedback entries reporting that weak-topic results feel accurate.

## Technical Metrics

- Availability: MVP uptime during user tests.
- Latency: diagnostic result generation time.
- Error rate: failed test submissions or path generation errors.
- Cost: AI generation cost per active learner when AI features are enabled.

## Review Cadence

Review metrics weekly during MVP validation.
