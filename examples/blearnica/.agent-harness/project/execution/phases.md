# Phases

## Phase 0: Foundation

Goal: set up the project foundation.

Tasks:

- Repository setup.
- Coding conventions.
- Basic data model.
- Authentication baseline if needed.
- Verification commands.

Exit criteria:

- Developers can run, test, and build locally.

## Phase 1: Core MVP

Goal: deliver the diagnostic-to-learning-path loop.

Tasks:

- Create learner profile.
- Create subjects and knowledge units.
- Build diagnostic test.
- Calculate diagnostic result.
- Generate learning path.
- Track progress.

Exit criteria:

- A learner can complete the loop end to end.

## Phase 2: AI Layer

Goal: add AI where it improves learning value.

Tasks:

- Explain diagnostic results.
- Recommend path adjustments.
- Generate practice tasks.
- Provide basic answer feedback.

Exit criteria:

- AI outputs are useful, reviewable, and bounded.
