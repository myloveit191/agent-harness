# MVP Scope

## Main Hypothesis

If learners know their weak knowledge units and receive a step-by-step learning
path, they will study more consistently and improve faster.

## Required Features

1. Diagnostic test.
2. Knowledge map.
3. Personalized learning path.
4. Practice tasks.
5. Progress tracking.

## Deferred Features

1. Mobile app.
2. Payment.
3. Social learning.
4. Advanced gamification.
5. Teacher marketplace.

## Success Criteria

- 20 users try the MVP.
- 60% complete the diagnostic test.
- 40% return within 7 days.
- At least 10 useful qualitative feedback entries.

## MVP Non-Goals

- Full course marketplace.
- Parent or teacher dashboard.
- Multi-subject content at launch.
