# User Stories

## Epic: Diagnostic Test

### Story 1

As a learner, I want to take an entry test so that I know my current level.

Acceptance criteria:

- The learner can start a diagnostic test.
- The test includes questions grouped by knowledge topic.
- The learner receives a result after completion.
- The result shows strengths and weaknesses.

### Story 2

As a learner, I want to see weak topics so that I know what to study first.

Acceptance criteria:

- Weak topics are shown in priority order.
- Each topic links to a learning path step.

## Epic: Learning Path

### Story 3

As a learner, I want a recommended path so that I can practice in a clear order.

Acceptance criteria:

- The path is based on diagnostic results.
- Each step contains practice tasks.
- Completion is recorded.
