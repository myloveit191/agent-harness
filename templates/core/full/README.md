# Full Core Template

This directory is an overlay on top of the MVP core.

Most files live inside the hidden `.agent-harness/` directory. If this folder
looks empty in Finder or another GUI, enable hidden files.

The full profile adds:

- Agent runtime and task routing guidance.
- Specs and plans for larger work.
- Evaluation checklists.

The lifecycle, gates, project memory, policies, superpowers, and verification
scripts come from the MVP core and are installed before this overlay.
