# MVP Core Template

This directory is the lightweight operating harness installed by default.

Most files live inside the hidden `.agent-harness/` directory. If you open this
folder in Finder or another GUI and only see `AGENTS.md`, enable hidden files.

The MVP core includes:

- Root `AGENTS.md` pointer.
- Lifecycle and workflow guidance.
- Project memory templates.
- Harness policies.
- Superpowers, MCP notes, progress files, and verification scripts.
